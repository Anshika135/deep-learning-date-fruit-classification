# Date Fruit Classification using Artificial Neural Network (ANN)

## 📌 Project Overview
This project builds a **Machine Learning classification model** using an **Artificial Neural Network (ANN)** to classify different varieties of **date fruits** based on dataset features.

The goal of this project is to demonstrate how deep learning models can be applied to agricultural datasets to accurately classify fruit types.

---

## 📊 Dataset
The dataset contains different **features extracted from date fruits** such as:

- Area
- Perimeter
- Major Axis Length
- Minor Axis Length
- Eccentricity
- Convex Area
- Extent
- Solidity
- Roundness
- Aspect Ratio
- and other morphological features

Each record belongs to a **specific date fruit variety**.

## 🧠 Model Architecture

The Artificial Neural Network consists of:

- Input Layer
- Hidden Layers with **ReLU activation**
- Output Layer with **Softmax activation** for classification

The model is trained using:

- **Categorical Crossentropy Loss**
- **Adam Optimizer**
